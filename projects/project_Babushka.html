Description

Babuska's Bullet is a 2D run and gun platformer. The object of the game is to defeat monsters and escape an evil mansion all while protecting your baby. The game has 3 distinct levels, multiple enemies, and a juicy platforming system. This game was designed and created in 36 hours at the 2019 Global Game jam. The theme was "what does home mean to you?". We drew inspiration from Yoshi's Island to make this game about protecting your child instead of worrying about getting hit by enemies. The goal was to foster feelings of protectiveness and tension toward the child. Play it here: https://kainkun.itch.io/babushka

How we built it

As you may have seen, this game was built with Unity. Fairly early on we knew we wanted to create a 2D platforming/shooting game. We used new 2D tools in Unity such as the tilemapper to quickly build levels for the game once we had the character controlller. The game makes heavy use of layering both in physics and rendering to help the game run well and look good.
