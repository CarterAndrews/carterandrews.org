Description

What is Crystal Craft. A virtual reality application for building atomic models. A tool for learning about crystal structures at the molecular level. A game for expressing creativity in a calm environment. A interdisciplinary, open source collaboration including students and faculty. and much more...

How we built it

Professor Wenwu Xu approached us with a need better teach the subject of crystal structures to his students. We met with him and discussed the plan going forward with the project. We decided on a virtual reality application where students could practice building their own crystal structures. Over the course of a semester we worked on the project and met with the professor to discuss goals. We used Unity3D, the steamvr framework, and a lot of custom code to put the project together. The project was tested with and eventually released to students for use in preperation for finals that semester.
